## 02 - Azure AI Foundry Lab Guide (35%)

Now shift from low-code configuration to model and prompt engineering decisions that improve outcome quality.

### Lab Goals

- Explore the **Model Catalog** with Phi-4 and Llama family models.
- Compare behavior using strong **System Prompts** in the Playground.
- Define a practical path to connect Foundry work back to **Copilot Studio**.

### Timebox

- Model catalog exploration: 12 min
- Prompt experimentation in Playground: 20 min
- Copilot Studio connection design: 10 min

### Step 1: Explore Model Catalog

1. Open **Azure AI Foundry** and navigate to **Model Catalog**.
2. Find a Phi-4 model and review core capabilities.
3. Find a Llama model and review core capabilities.
4. Capture a quick decision note: which model suits your use case and why.

### Step 2: Build and Compare System Prompts in Playground

1. Open **Playground**.
2. Write a short system prompt with role, constraints, and expected tone.
3. Test the same user prompt against two different models.
4. Evaluate output quality on relevance, safety, and actionability.
5. Refine your system prompt and run one final test.
