## AI for Good Inspiration

Use these scenarios to help teams pick a focused challenge quickly.

### 1. Health Access Navigator
Build an agent that helps people find nearby clinics, eligibility requirements, and next steps for basic services.

### 2. Climate Action Coach
Create an agent that translates local environmental data into practical household and community actions.

### 3. Learning Support Companion
Design an agent that gives students personalized study plans, encouragement, and local learning resources.

### 4. Crisis Information Assistant
Prototype an agent that shares verified emergency guidance and routes high-risk cases to human responders.

### 5. Community Services Matcher
Build an agent that helps residents discover food, housing, and employment support programs in their area.

### Prompt Starters for Teams

- "Who is the user and what high-impact decision do they need to make quickly?"
- "What trusted sources will ground the agent's answers?"
- "Where should the agent escalate to a human?"
